# Predictive Equipment Condition Monitoring Dashboard

**Skills Demonstrated:** Data Analysis, Dashboard Development, KPI Reporting, Trend Analysis, Data Visualization, Business Intelligence

## Project Overview

This project analyzes simulated equipment performance data to identify maintenance trends, equipment failures, and operational risks. The goal was to demonstrate how data analytics can support predictive maintenance and reduce equipment downtime.

---

## Business Problem

Unexpected equipment failures can lead to increased repair costs, production delays, and lost revenue. This dashboard helps identify warning signs before critical failures occur.

---

## Tools Used

- Microsoft Excel
- Data Visualization
- Dashboard Design
- Data Analysis

---

## Key Metrics

- Oil Temperature
- Coolant Temperature
- Downtime
- Fault Severity
- Repair Costs
- Engine Hours

---

## Key Insights

- Rising oil temperatures frequently occurred before critical equipment failures.
- Higher downtime was associated with significantly increased repair costs.
- Equipment with recurring warning faults showed a greater likelihood of progressing to critical failures.

---

## Business Impact

This dashboard demonstrates how data visualization can support proactive maintenance planning, reduce operational risk, and improve equipment reliability.
